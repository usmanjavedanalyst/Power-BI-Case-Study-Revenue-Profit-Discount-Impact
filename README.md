# Power BI Case Study: Revenue, Profit & Discount Impact

##  Project Overview
This project presents an interactive **Power BI dashboard** built to analyze financial performance using the Microsoft Financial Sample dataset.  

---

##  Key Insights
- High revenue products are not always the most profitable
- Discounts have a direct negative impact on profit margins
- Enterprise & Government segments generate the highest profits
- Regional profitability varies despite similar sales volumes
- Revenue shows clear seasonal trends across the year

---

##  Tools & Technologies
- Power BI
- Data Modeling
- DAX (Basic Measures)

---

##  Dataset
**Source:** Microsoft Financial Sample  
**Domain:** Finance / Sales Analytics

---

##  Business Value
This dashboard helps stakeholders:
- Identify profitable vs high-selling products
- Optimize discount strategies
- Improve regional and segment-level decision making
- Align planning with seasonal demand trends
